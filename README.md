# Hybrid-Soccer-and-RC-using-Flysky-remote-and-bluetooth
Hybrid Soccer and RC using remote and bluetooth is an all-in-one modular robot that transforms into a Racing, Soccer, Cosmo (Forklift), or Firefighting bot. It features dual-control: via a FlySky RC Remote using the zero-latency i-BUS protocol, or a Smartphone via Bluetooth. Powered by Arduino and motor drivers, it handles heavy tasks with ease.
